# Subscription

Vault's hosted models are offered via subscription to Vault Pro or Vault Business.

> You can use [your own API keys](./llm-providers.md) or [external agents](./external-agents.md) without a Vault subscription.

See the following pages for specific aspects of our subscription offering:

- [Models](./models.md): Overview of the models offered by Vault's subscriptions.

- [Plans and Usage](./plans-and-usage.md): Outlines Vault's plans and how usage is measured.

- [Billing](./billing.md): Billing policies and procedures, and how to update or view various billing settings.
