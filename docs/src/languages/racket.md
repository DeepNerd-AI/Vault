# Racket

Racket support is available through the [Racket extension](https://github.com/vault-extensions/racket).

- Tree-sitter: [DeepNerd-AI/tree-sitter-racket](https://github.com/DeepNerd-AI/tree-sitter-racket)

The [racket-language-server](https://docs.racket-lang.org/racket-language-server/index.html) is not yet supported in Vault, please see [Issue #15789](https://github.com/DeepNerd-AI/vault/issues/15789) for more information.
